# Caesar Cipher

Two command-line tools for working with Caesar ciphers.

## caesar.py

Encrypts or decrypts text using a Caesar cipher with a given shift number. Handles both uppercase and lowercase letters.

```
python caesar.py
```

You'll be prompted to choose encrypt (e) or decrypt (d), then enter your text and shift number.

## boudica.py

Brute-force codebreaker. Takes a ciphertext and shows the first 20 characters of all 25 possible decryptions. You pick the one that looks like real text, and it decrypts the full message.

```
python boudica.py
```
